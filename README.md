🧮 JavaScript Calculator
A functional calculator built using HTML, CSS, and Vanilla JavaScript, focused on learning programming logic and state management.

🚀 Features


✅ Basic operations:
+, -, *, /


✅ Dynamic number input


✅ Decimal support (.)


✅ Special buttons:

C → clear all
CE → clear current input
% → percentage calculation
+/- → toggle positive/negative
= → calculate result



✅ Repeated operation with =
(e.g. 2 + 3 = 5, = → 8, = → 11)


✅ Operator replacement before entering the second number
(e.g. 2 + → change to -)


✅ Error handling:

division by zero




🧠 Concepts Applied
This project uses core programming concepts:


State management:

current → current number
previous → previous number
operator → current operator



Operation memory:

lastOperator
lastOperand



Conditional logic (if, else)


DOM manipulation


Separation of concerns (calculate() function)
